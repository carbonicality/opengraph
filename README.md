# opengraph 
opengraph is a cool lil graphing calculator inspired by Desmos, built with HTML, CSS and JS.
it utilises mathlive and has loads of cool features, like function plotting, basic polar equations, explicit and implicit equations plotting too

## features
- graphing (shocking)
- intersection indicators
- history
- latex and math live
- cool virtual keyboard using mathlive
- x and y intercept indicators
- trace mode
- pan and zoom
- see x y and intersection tooltips by hovering over them so tooltip stuff
- function labels over lines
- line colour picker
- line styles picker
- parameters 
- table of values
- data exports
- screenshotting graph
- grid lines toggling
- saving and loading
- cool ui!

## usage
open the website, and put your equations in the left hand sidebar!
press the keyboard button to open the virtual keyboard
use the + button or the faded box to add new equations
